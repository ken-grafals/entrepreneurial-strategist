# Decision Log

A chronological record of the decisions made in this project — the reasoning, the evidence, and the files that resulted. New entries go at the bottom.

Entry format:

```
## YYYY-MM-DD — <short decision title>

**Decision:** <one sentence>
**Reasoning:** <2–4 sentences grounded in specific artifacts and founder context>
**Artifacts:**
- <file path>
- <file path>
**Kill / reversal conditions:** <specific, falsifiable conditions that would cause this decision to be revisited>
```
